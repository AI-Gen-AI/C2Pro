# ztwq-mvp

ZTWQ es un prototipo que demuestra un auditor inteligente capaz de comparar contratos con cronogramas de proyecto.

## Instalacion

1. Crea un entorno virtual e instala dependencias (por definir).
2. Define la variable `OPENAI_API_KEY` en un archivo `.env`.

## Ejecucion rapida

```
python backend/auditor_ia.py
```

## Estructura del repositorio

- `backend/` - Modulos de procesamiento y acceso a la API de IA.
- `frontend/` - Interfaz (Streamlit) a desarrollar.
- `data_samples/` - Documentos de prueba.
- `tests/` - Pruebas automatizadas.
